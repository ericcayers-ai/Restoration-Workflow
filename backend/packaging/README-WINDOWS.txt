Restoration Workflow — Windows portable package
================================================

1. Extract this whole folder anywhere (Desktop, Documents, etc.).
2. Double-click Run.bat
3. Your browser opens the app at http://127.0.0.1:8765

No Python install is required. A GPU is optional — CPU works, just slower.
Model weights download only when you use a model (Settings → Manage Downloads,
or the first Simple Mode run).

Advanced: you can also launch RestorationWorkflow.exe directly.
Close the Run.bat console window to stop the server.
