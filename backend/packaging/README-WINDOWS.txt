Restoration Workflow — Windows install
======================================

This folder is installed by the Windows Setup program.

1. Launch "Restoration Workflow" from the Start Menu (or Desktop shortcut).
2. Your browser opens the app at http://127.0.0.1:8765

You can also double-click Run.bat or RestorationWorkflow.exe in this folder.
Close the console window to stop the server.

No Python install is required. A GPU is optional — CPU works, just slower.
Model weights download only when you use a model (Settings → Manage Downloads,
or the first Simple Mode run).
